Linea de comandos Linux

  • File system
    1. /root: directorio con todos los privilegios del usuario root.
    2. /etc: contiene generalmente los archivos de configuracion, cuando y como inician los programas
    3. /home: directorio principal del usuario.
    4. /mnt: directorio donde se montan archivos manualmente.
    5. /media: se suelen montar CD y dispositivos extraibles como USB.
    6. /bin: directorio de aplicaciones binarias importantes(el equivalente a ejecutables en Microsoft).
    7. /lib: directorio donde se encuentran todas las librerias del sistema(programas que son similares a los DLL de Microsoft).
    8. /proc: directorio que ofrecer informacion sobre el sistema cerca de los preocesos
    9. /sys: contiene archivos del sistema .
  • pwd: directorio de trabajo actual.
  • man <app/command>: comando incorporado, da acceso a las páginas del manual para la mayoría de las herramientas.
  • su : permite cambiar de usuario sin cerrar sesión y volver a iniciarla de la sesión primaria.
  • whoami: visualiza que el nombre de usuario actual.
  • ls: listar todos los archivos de directorio actual
    1. -a: muestra todos los archivos/directorios incluidos los “.”
    2. -l: muestra los archivos en formato de lista
  • cd: cambio de directorio.
  • cat <name file>: concatenar archivos lo que proporciona una salida del contenido de cada archivo .
  • find: permite buscar cualquier archivo en el sistema.
    1. /<directorie>: enumerar todos los archivos en el directorio actual.
    2. /: enumera todos los archivos del sistema operativo.
    3. dir -user <-user>: enumera todos los archivos de un usuario especifico.
    4. dir -group <-group user >: enumera todos los archivos de un grupo de usuario especifico.
    5. -perm: encuentra todos los archivos con permisos especificos.
  • apt-get <command>: comando en sistemas derivados de debian para instalar programas en Linux.
    1. list : lista los paquetes según los nombres
    2. search: busca en las descripciones de los paquetes
    3. show: muestra detalles del paquete
    4. install: instala paquetes
    5. remove: elimina paquetes
    6. autoremove: Elimina automáticamente todos los paquetes sin utilizar
    7. update: actualiza la lista de paquetes disponibles
    8. upgrade: actualiza el sistema instalando/actualizando paquetes
    9. full-upgrade: actualiza el sistema eliminando/instalando/actualizando paquetes
    10. edit-sources: edita el fichero de información de fuentes
    11. purge <app > : eliminar aplicación
    12. update –fix-missing: error al actualizar los paquetes
  • wget: descargardor de archivos no interactivo
  • Man <command>: comando incorporado, da acceso a las páginas del manual para la mayoría de las herramientas.
