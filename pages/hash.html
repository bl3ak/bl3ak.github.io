    4. File 1 local_passwd contiene: root:x:0:0::/root:/bin/bash
    5. File 2 local_shadow contiene: root:$6$2nwjN454g.dv4HN/$m9Z/ r2xVfweYVkrr.v5Ft8Ws3/ YYksfNwq96UL1FX0OJjY1L6l.DS3KEVsZ9rOVLB/ ldTeEL/OIhJZ4GMFMGA0: 18576::::::
